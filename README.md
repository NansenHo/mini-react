# Mini React

实现一个简化版 React 模型。

## Quick Start

```bash
git clone https://github.com/NansenHo/mini-react.git
cd mini-react

pnpm i

pnpm dev

pnpm test
```

## Tech Stack

- Vite
- Vitest
- TypeScript
