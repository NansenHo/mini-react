# Mini React 18

このリポジトリは React 18 の学習を深めるために、React 18 の簡略版を実装しています。

## 🛠️ クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/NansenHo/mini-react.git
cd mini-react

# パッケージをインストール
pnpm i

# 開発環境を起動
pnpm dev

# 単体テスト
pnpm run test-unit

# E2E テスト
pnpm run test-e2e

# ビルド
pnpm run build
```

- Vitest
