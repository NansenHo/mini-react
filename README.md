# Mini React

Implementing a minimal React model.

## Quick Start

```bash
git clone https://github.com/NansenHo/mini-react.git
cd mini-react

pnpm i

pnpm dev

pnpm test
```

## Tech Stack

- Vite
- Vitest
- TypeScript
